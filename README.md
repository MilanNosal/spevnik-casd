# Spevník CASD

An iOS songbook app for the Church of Adventists of the Seventh Day (Cirkev adventistov siedmeho dňa) hymnal *Spievajme Hospodinovi*.

Songs are bundled as JSON (`spevnik-casd/Resources/spevnik.json`), seeded into a
[SwiftData](https://developer.apple.com/documentation/swiftdata) store on first
launch, and re-seeded whenever the app build number changes. Browse the numbered
list, then swipe through songs in a full-screen pager.

## Requirements

- Xcode 15 or later
- iOS 18 or later (SwiftData)
